## 学习小组形式
**我们的活动并不是讲座形式，需要大家积极参与讨论。**

1. 每周学习讨论一个深度学习的主题
2. 每次讨论会一个半小时
3. ** 半小时报告，一个小时讨论或互相学习 **
4. 报告人提出三到五个问题带大家讨论
